# HW2 Project Write-up

## 项目题目

谁真正实现了“经济增长与碳排放脱钩”？

## 作品试图回答的问题

哪些国家在 1990 年之后真正实现了经济增长与碳排放脱钩？这种变化是短期波动，还是伴随着能源结构转型的长期趋势？

因此，作品采用“总览 - 细看 - 解释”的结构：
- 先用象限散点图判断国家位于哪个增长 / 排放象限
- 再用历史路径图观察它是如何移动到当前位置的
- 最后用能源结构视图解释这种变化背后的原因

## 设计决策

- 总览使用象限散点图，因为“增长 / 排放”两个维度构成了最直接的脱钩判断空间。
- 总览图支持故事注释、详细 tooltip 和多国选择，便于快速讲清关键国家。
- 历史路径视图保留轨迹图与折线图双模式，兼顾单国叙事和多国比较。
- 能源结构视图保留趋势图、起终点对比和多国比较三种表达，避免只看单一国家。
- 页面加入中英双语切换，便于课堂展示与后续公开展示。

## 考虑过的替代方案

- 考虑过直接用世界地图做总览，但地图对“增长与排放变化”的二维关系表达不如散点图直接。
- 考虑过只做折线图 dashboard，但这样不容易快速判断谁真正位于脱钩区域。
- 考虑过只保留能源堆叠图，但多国比较时信息会变得混乱，所以补充了起终点对照和结构比较表。

## 数据来源

- Our World in Data CO2 Data  
  提供 CO2、CO2 per capita、consumption CO2、GDP、population 等指标。
- Our World in Data Energy Data  
  提供 renewables、coal、gas、oil、fossil share 等能源结构指标。
- World Bank Country Metadata API  
  补充地区和收入水平信息。

## 数据处理与实现方式

- 使用官方数据源下载原始 CSV 与国家元数据。
- 预处理脚本将多个来源按国家与年份对齐，生成前端直接可用的国家-年份数据。
- 为保证 GDP 覆盖稳定，当前默认终点年份使用 2022。
- 前端使用 Vue + Vite + D3 实现，数据以静态 JSON 方式加载。

## 页面结构

1. 顶部导语与故事模式按钮  
   用于快速切入几个课堂展示最有代表性的国家组合。
2. Sticky 控制区  
   包含搜索、地区筛选、收入筛选、指标切换与时间刷选。
3. Overview Scatter  
   用于判断哪些国家真正实现了“高增长 / 低排放”。
4. Trajectory  
   用于观察国家历史路径，并支持 hover 高亮与图例锁定。
5. Insight Panel  
   自动生成当前国家的解释性文字，降低阅读门槛。
6. Energy Mix  
   用于解释脱钩背后的能源结构变化，并支持多国比较。
